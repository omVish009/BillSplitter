# BillSplitter

## Overview
BillSplitter is a simple project that helps users split bills easily among friends, family, or group members.

The main goal of this project is to calculate how much each person needs to pay when a bill is shared.

---

## Features
- Add total bill amount
- Add number of people
- Split bill equally
- Easy and simple calculation
- User-friendly project
- Useful for restaurants, trips, and group expenses

---

## Tech Stack
- Java
- OOP Concepts
- Basic Logic Building

---

## How It Works
1. User enters the total bill amount
2. User enters the number of people
3. System calculates each person's share
4. Final split amount is displayed

---

## Example
Total Bill: 1000  
Number of People: 5  

Each Person Pays: 200
